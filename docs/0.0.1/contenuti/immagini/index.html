Immagini

Documentazione ed esempi per l’inserimento di immagini responsive (che quindi non diventano mai più grandi dei loro elementi genitore), aggiungendo loro stili non invasivi.

Immagini responsive

Le immagini in Bootstrap Italia sono rese responsive con .img-fluid. max-width: 100%; e height: auto; vengono applicati all’immagine in modo che sia ridimensionata attraverso l’elemento padre.

Thumbnails di immagini

Oltre alle nostre utilità sul border-radius, puoi usare .img-thumbnail per dare a un’immagine un aspetto del bordo arrotondato di 1 pixel.

Allineare le immagini

Puoi allineare le immagini con le classi di supporto sul float o con le classi sul text alignment. Le immagini a livello block possono essere centrate usando la classe di utilità margin .mx-auto.

Picture

Se stai usando l’elemento <picture> per specificare multipli elementi <source> per una specifica <img>, assicurati di aggiungere la classe .img-* al tag <img> e non al tag <picture>.

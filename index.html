Bootstrap Italia è il modo più semplice per costruire interfacce inclusive e semplici da mantenere. È una libreria completamente open-source costruita sulle fondamenta di Bootstrap {{ site.bootstrap_version }}, la libreria più conosciuta per la creazione di applicazioni web responsive in modo facile e sicuro.

Bootstrap Italia eredita tutte le funzionalità, componenti, griglie e classi di utilità già presenti nella versione ufficiale di Bootstrap, e le personalizza secondo le Linee Guida di Design per i siti web della Pubblica Amministrazione.

Attenzione

La libreria è tuttora in fase di sviluppo e alcuni componenti potrebbero non essere ancora presenti o incompleti. È possibile vedere lo stato di ogni componente nel menu principale attraverso il tag wip.

Bootstrap Italia

Bootstrap Italia è una libreria completamente open-source costruita sulle fondamenta di Bootstrap {{ site.bootstrap_version }} per la creazione di applicazioni responsive, mobile-first per il web in modo semplice e sicuro.

Bootstrap Italia eredita tutte le funzionalità, componenti, mixins, grid system, ed altro già presenti nella versione ufficiale di Bootstrap, e le personalizza secondo le linee guida di design per i siti web delle Pubbliche Amministrazioni.

Attenzione

La libreria è tuttora in fase di sviluppo e alcuni componenti potrebbero non essere ancora presenti o incompleti. È possibile vedere lo stato di ogni componente nel menu principale attraverso il tag wip.

Collabora con noi!

Il codice e la roadmap di Bootstrap Italia sono pubblici e in continua evoluzione. All'interno del repository di Bootstrap Italia puoi trovare maggiori informazioni su come contribuire o come compilare il codice sorgente per personalizzare la libreria, sistemare bug o creare nuovi componenti!
